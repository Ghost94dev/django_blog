# 📝 Django Blog

A simple **multi-author blog application** built with Django. Users can create, edit, and share blog posts with others.

## 🚀 Features
- User authentication
- Create, edit, and delete posts
- Rich text formatting
- Multi-author support
- Responsive UI

## 🛠 Tech Stack
- Django (Python)
- HTML, CSS
- SQLite (default)
